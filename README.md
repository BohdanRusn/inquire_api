## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.


### For the program to work, you need to change the parameters in the .env file to your database.

By default, I left my credentials.

PG_HOST - Postgres host

PG_PORT - Postgres port

PG_USER - Postgres user name

PG_PASSWORD - Postgres user password

PG_DB - Postgres DB name

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
